# Assign max trip duration

Check and convert the max trip duration input.

## Usage

``` r
assign_max_trip_duration(
  max_trip_duration,
  modes,
  max_walk_time,
  max_bike_time
)
```

## Arguments

- max_trip_duration:

  A number.

## Value

An `integer` representing the maximum trip duration in minutes.
