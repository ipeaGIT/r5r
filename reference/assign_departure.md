# Check and convert POSIXct objects to strings

Check and convert POSIXct objects to strings

## Usage

``` r
assign_departure(datetime)
```

## Arguments

- datetime:

  An object of POSIXct class.

## Value

A list with the `date` and `time` of the trip departure as characters.
