# Constructor for r5r_network object

Wraps a Java R5RCore as an r5r_network.

## Usage

``` r
wrap_r5r_network(jcore)
```

## Arguments

- jcore:

  A `jobjRef` Java object reference to R5RCore.

## Value

`r5r_network`
